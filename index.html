SplitWisely

Expense Splitting Android Application

Features

  • User Authentication: Secure sign-up and login using Firebase Authentication.
  • Expense Management: Add expenses with details like amount, description, date, and category.
  • Group Functionality: Create groups to manage shared expenses among friends or family.
  • Real-time Sync: All data is synced in real-time across devices using Firebase Firestore.
  • Balance Calculation: Automatically calculates who owes whom and displays balances.
  • User-friendly Interface: Intuitive and easy-to-navigate UI for seamless user experience.
